# 基础模型路由

本文件只做“论文级基础分析”的模型选择，不把高级方法包装成默认方案。每次选择模型时，必须同时记录：因变量类型、数据结构、识别目标、标准误层级、解释口径。

## 1. 快速决策树

1. 先判断因变量：连续、二元、有序、多分类、计数、持续时间、文本/编码结果。
2. 再判断数据结构：横截面、重复横截面、面板、层级数据、配对/聚类样本。
3. 再判断研究目标：描述相关、理论检验、机制检验、政策评估、预测分类。
4. 最后选择报告口径：系数、边际效应、预测概率、IRR、组间差异、主题频次。

如果研究设计不能支持因果解释，模型名称不得把相关关系写成“影响”或“效应”。

## 2. 因变量类型

| Y 类型 | 首选模型 | 必报 | 常见误区 |
|---|---|---|---|
| 连续 | OLS；有组内结构时 FE OLS | 系数、t 统计值、N、R2/组内 R2 | 把单位不清的系数写成 substantive effect |
| 二元 | logit/probit；线性概率作稳健性 | AME 或预测概率，不能只报 logit 系数 | 用 odds ratio 替代机制解释 |
| 有序 | ordered logit/probit；必要时广义 ordered | 各类别边际效应或关键类别概率变化 | 只解释潜变量系数 |
| 多分类 | multinomial logit；基准类别明确 | 预测概率或相对风险比 | 不说明参考类别 |
| 计数 | Poisson；过度离散时 NB | IRR 或边际效应，过度离散诊断 | 只因过度离散就放弃 Poisson 稳健性 |
| 比例/率 | fractional logit；OLS 作可读性补充 | 预测比例变化 | 忽略 0/1 边界 |
| 面板连续 | FE/RE/TWFE | id/time、组内变化来源、聚类 SE | 不说明固定效应吸收了什么 |
| 文本/编码 | 频次、比例、主题模型或人工编码统计 | 抽样单位、编码单位、信度 | 把 LLM 标注当作人工编码替代 |

## 3. 研究设计路由

| 设计 | 基础实现 | 必要诊断 | 稳健性 |
|---|---|---|---|
| 描述相关 | M1 双变量、M2 控制变量、M3 固定效应 | 变量分布、缺失、异常值、共线性 | 替代样本、替代口径、替代控制组 |
| 理论检验 | 理论核心 X + 竞争解释控制 | 理论变量是否先于结果、测量效度 | 替代理论解释、替代测量 |
| DiD | treatment × post；事件研究 | 平行趋势、处理时点、样本窗口 | 安慰剂、替代窗口、组别趋势 |
| IV/2SLS | 第一阶段 + 第二阶段 | 第一阶段 F、排除限制论证 | 过识别、弱工具、缩减式 |
| RD | running variable + cutoff | 带宽、操纵检验、协变量平衡 | 替代带宽、核函数、placebo cutoff |
| PSM/加权 | 倾向得分 + 匹配/加权后回归 | 平衡性、共同支撑 | 匹配方法、卡尺、IPW/entropy balancing |
| 中介 | 理论中介链 + 间接效应 | 时间顺序、混杂风险 | 替代中介、反向路径、bootstrap |
| 异质性 | 交互项或分组回归 | 分组理论依据、样本量 | 边际效应图、组间差异检验 |
| 机制检验 | 中介、过程证据、替代解释排除 | 机制变量是否贴近理论 | 多指标机制、质性证据互证 |

## 4. 标准误选择

- 横截面普通回归：默认 HC3；如果抽样有聚类，按抽样/处理层级聚类。
- 个体、学校、企业、地区内相关：按相关性发生的最高合理层级聚类。
- 面板数据：通常按个体或政策实施层级聚类；双向相关时考虑双向聚类。
- DiD：按处理实施层级聚类；组数太少时说明限制，可补 wild cluster bootstrap。
- 多层数据：若研究问题关注组间差异，考虑 multilevel model；否则至少聚类 SE。
- 权重数据：明确权重类型，区分抽样权重、频数权重和分析权重。

## 5. 表格顺序

常用回归表：

1. M1：核心解释变量，展示基线关系。
2. M2：加入个体/家庭/组织控制。
3. M3：加入地区/年份/组织固定效应。
4. M4：机制、交互项或异质性。

识别设计表格：

1. 诊断表：平衡性、第一阶段、平行趋势或操纵检验。
2. 主结果表：核心识别模型。
3. 稳健性表：替代样本、口径、窗口、标准误。
4. 机制/异质性表：只放理论驱动结果，避免撒网式展示。

## 6. 诊断阈值速查

回归后必须按因变量类型和数据结构的适用性选择性做诊断，不做机械全跑：

| 检验目标 | 方法 | 阈值/标准 | 适用场景 |
|---------|------|----------|---------|
| 共线性 | VIF | > 10 严重；> 5 关注 | 所有 OLS/线性模型 |
| 异方差 | Breusch-Pagan / White | p < 0.05 | 截面 OLS（使用 robust SE 前确认） |
| 自相关 | DW / Breusch-Godfrey | DW≈0或4；BG p<0.05 | 时间序列、面板 |
| 弱工具变量 | Cragg-Donald Wald F / Kleibergen-Paap F | F < 10 弱 IV | IV/2SLS |
| 过度识别 | Sargan / Hansen J | p < 0.05 拒绝外生性 | 多工具变量时 |
| 内生性 | Durbin-Wu-Hausman | p < 0.05 存在内生性 | OLS vs IV 对比 |
| FE vs RE | Hausman | p < 0.05 → FE | 面板模型选择 |
| 截面相关 | Pesaran CD | p < 0.05 | 长面板 (N小T大) |
| 面板自相关 | Wooldridge (xtserial) | p < 0.05 | 面板数据 |
| 正态性 | Shapiro-Wilk | p < 0.05 | 小样本 OLS 残差 |
| 平行趋势 | 事件研究法 pre-trend 联合检验 | p > 0.05 (处理前不显著) | DID |
| RDD 操纵 | McCrary 密度检验 | p > 0.05 (断点处无跳跃) | RDD |
| 平衡性 | pstest 标准化偏差 | 匹配后 |bias| < 10% | PSM |
| GMM 自相关 | AR(2) 检验 | p > 0.05 (无二阶自相关) | 动态面板 GMM |
| GMM 过度识别 | Hansen test | p > 0.05 | 系统/差分 GMM |

诊断不通过的处理顺序：(1) 确认数据清洗无误；(2) 调整模型设定（加控制变量/固定效应/标准误类型）；(3) 换用更适用的估计方法；(4) 在论文中如实报告并讨论限制。

## 7. 内生性判断框架

遇到以下任一情况，必须考虑内生性：
- 遗漏变量同时影响 X 和 Y（如能力影响教育和收入）
- X 和 Y 互为因果（如价格和销量）
- 核心解释变量存在测量误差（如自报收入）

解决路径优先级：面板固定效应（去时间不变遗漏变量）→ 工具变量（需要有效 IV）→ DID/RDD（需要制度/政策断点）→ PSM（仅解决可观测选择）。

IV 的三条件必须同时满足：
1. **相关性**：Corr(Z, X) ≠ 0，第一阶段 F > 10
2. **外生性**：Corr(Z, ε) = 0，不能直接检验，必须理论论证
3. **排他性约束**：Z 只能通过 X 影响 Y

弱 IV (F<10) 时，2SLS 的偏差可能比 OLS 更严重（向 OLS 偏倚），改用 LIML。

## 8. 面板模型选择逻辑

```
面板数据 → 混合OLS vs FE (F test, p<0.05 → 拒绝混合OLS)
         → FE vs RE (Hausman: p<0.05 → FE; p>0.05 → RE更有效)
         → 含因变量滞后 → 动态面板偏差 (Nickell bias)
            → T 小 → Diff-GMM / Sys-GMM
            → T 中等 → 偏差校正 LSDV (xtlsdvc)
         → N小T大 → 长面板策略
            → DK标准误 (xtscc)、PCSE (xtpcse)、FGLS (xtgls)
```

模型选择必须说明：(1) 固定效应吸收了哪些维度的不可观测异质性；(2) 组内变异是否足够识别核心系数；(3) 为什么该聚类层级合理。

## 9. 空间依赖判断框架

遇到以下情况，考虑空间计量：
- 观测单位存在地理邻接或距离关系
- 理论预期存在空间溢出/集聚效应
- OLS 残差的 Moran's I 显著

处理流程：OLS → estat moran（诊断空间自相关）→ 显著则构造 W 矩阵 → SDM（最通用）→ 效应分解（直接/间接/总效应）。

空间计量必须报告：权重矩阵类型（邻接/反距离/经济距离）、行标准化方式、莫兰指数及其显著性。

## 10. 输出记录

模型选择必须写入 `analysis-output/reports/model-decision-[date].md` 或过程日志，至少包含：

- 为什么该模型匹配因变量和数据结构。
- 为什么该标准误层级合理。
- 哪些结果可以因果解释，哪些只能描述相关。
- 哪些稳健性是必须项，哪些是可选项。
- 使用 IV/DID/RDD 等识别策略时，必须说明识别假设及其可信度论证。
