# 基础模型路由

## 1. 因变量类型

| Y 类型 | 首选模型 | 必报 |
|---|---|---|
| 连续 | OLS / FE OLS | 系数、SE、N、R2 |
| 二元 | logit/probit | AME 或预测概率、AUC 可选 |
| 有序 | ordered logit/probit | 各类别边际效应 |
| 多分类 | multinomial logit | 预测概率或相对风险比 |
| 计数 | Poisson；过度离散时 NB | IRR 或边际效应 |
| 面板连续 | FE/RE/TWFE | id/time、Hausman、聚类 SE |

## 2. 研究设计

| 设计 | 基础实现 | 稳健性 |
|---|---|---|
| 描述相关 | M1 双变量、M2 控制变量、M3 固定效应 | 替代样本、替代口径 |
| DiD | treatment x post；事件研究 | 平行趋势、安慰剂、替代窗口 |
| IV/2SLS | 第一阶段 + 第二阶段 | 第一阶段 F、过识别、弱工具 |
| PSM | 匹配后均值差/回归 | 平衡性、匹配方法 |
| 中介 | 分步回归或 bootstrap 间接效应 | 替代中介、反向解释 |
| 异质性 | 交互项或分组回归 | 边际效应图、组间差异检验 |

## 3. 标准误选择

- 横截面普通回归：HC3 或 robust。
- 个体、学校、企业、地区内相关：按聚类层级 cluster。
- 面板数据：通常按个体或政策实施层级 cluster。
- DiD：按处理实施层级 cluster；组数太少时报告限制。

## 4. 表格顺序

常用回归表：

1. M1：核心解释变量。
2. M2：加入个体/家庭/组织控制。
3. M3：加入地区/年份/组织固定效应。
4. M4：交互项、机制或异质性。

非描述性设计不强制 M1-M4 阶梯；DiD、IV 等以识别策略为主。
