# 结果报告规范

本文件规定 analysis 模块产物如何进入论文正文。核心原则：先解释模型和样本，再报告方向、量级和不确定性，最后说明理论含义与边界。数值格式全文统一。

---

## 1. 数值与格式规范

### 1.1 小数位规则

| 统计量 | 小数位 | 示例 |
|--------|--------|------|
| 描述统计（均值/SD/min/max） | 3 | 0.387 (0.152) |
| 回归系数、边际效应 | 3 | 0.234*** |
| t/z 统计值 | 3 | (3.421) |
| 比例/百分比（表格） | 3 | 0.234 |
| 百分比（正文） | 1 | 23.4% |
| p 值 | 3 或 exact | p = 0.032 |
| R² / pseudo-R² | 3 | 0.287 |
| 样本量 N、频数、自由度 | 0 | N = 12,847 |
| F 统计量 | 2 | F = 23.45 |

### 1.2 显著性标注

```
* p < 0.05, ** p < 0.01, *** p < 0.001
```

统一使用三星标注。

### 1.3 括号使用

| 场景 | 格式 |
|------|------|
| 系数 + 统计值 | 0.234*** (3.421) |
| 带 CI | 0.234 [95% CI: 0.100, 0.368] |
| 表注 | 括号内为 t 统计值 |
| 正文 | "系数为 0.234（t = 3.42, p = 0.001）" |

## 2. 描述统计报告

Table 1 必须包含：

- 全样本的描述统计（均值、SD、min、max、N）
- 如有关键分组（处理组/控制组），附加分组均值 + 差异检验
- 样本筛选流程（如从原始数据到分析样本的流失表）作为附录或正文脚注

## 3. 回归结果报告规范

### 3.1 主回归表 (Table 2)

每列报告一个模型的系数和 t/z 统计值，结构为逐步加入变量/固定效应：

| 列 | 内容 |
|----|------|
| (1) | 仅核心解释变量 |
| (2) | + 个体/家庭层面控制变量 |
| (3) | + 地区/行业/时间固定效应 |
| (4) | 如有 DID/IV/RDD，为核心识别模型 |

**全部控制变量均须在表中逐行列报**，不得选择性展示或仅以"控制变量已控制"一笔带过。

## 5. 稳健性报告规范

### 5.1 稳健性检验的论文组织结构

稳健性不在正文中逐一赘述，而是结构化呈现：

1. **正文**：概述稳健性策略（"本文进行了 X 类稳健性检验：替换变量、替换样本、..."），报告核心结论的稳健性方向
2. **附录表**：每项稳健性一列，集中在一个或多个 Appendix table
3. **不可选择性报告**：所有做了的稳健性必须报告，无论是否支持主结论

### 5.2 稳健性表模板

```
附录表 A3：稳健性检验

           (1)替代Y   (2)替代X   (3)子样本   (4)无FE    (5)替代SE
───────────────────────────────────────────────────────────────
X            0.187**    0.213***   0.234**    0.312***   0.234**
            (2.876)    (3.098)    (2.987)    (3.872)    (2.543)
───────────────────────────────────────────────────────────────
注：(1)替换因变量为[Y_alt]；(2)替换核心解释变量为[X_alt]；
(3)剔除[某群体]后的子样本；(4)移出固定效应；(5)按[alt_cluster]聚类。
所有模型中核心系数的方向和显著性均与基准结果一致。
```

### 5.3 稳健性表述模板

结果稳健时：
> "表 A3 报告了一系列稳健性检验。替换因变量口径、替换核心解释变量、限制子样本、调整固定效应以及改变标准误聚类层级后，核心系数的方向和显著性均未发生本质变化，说明基准结论具有较好的稳健性。"

结果部分稳健时：
> "尽管基准结果的方向和显著性在多数稳健性检验中保持一致，但在 [某特定检验] 中系数降至 [值] 且不再显著，提示 [可能原因]。"

## 6. 图形规范

### 6.1 通用要求

| 要素 | 规范 |
|------|------|
| 标题 | 图形标题放置在图形下方（figure caption），不以"图X："开头 |
| 坐标轴标签 | 含变量名和单位，英文或中英文对照 |
| 字号 | 坐标轴标签 ≥ 8pt，标题 ≥ 10pt |
| 配色 | 色盲友好的调色板（viridis / ColorBrewer）；不用红绿对比 |
| 分辨率 | 导出时 ≥ 300 DPI |
| 格式 | 论文投稿用 PDF（矢量）；预览/Word 嵌入用 PNG |
| 比例 | 宽高比约 1.4:1（默认 7×5 inch） |
| 网格线 | 浅灰色、虚线或半透明，不抢夺数据视觉权重 |
| 图例 | 放在图内空白处或下方，不加外框 |
| 背景 | 白色或无背景，不用灰色/彩色背景 |

### 6.2 常用图类型

| 图类型 | 用途 | 格式要求 |
|--------|------|---------|
| 分布直方图 | 展示因变量或关键变量分布 | 加密度曲线、标注均值/中位数 |
| 散点图 + 拟合线 | 展示双变量关系 | 加 95% CI 阴影 |
| 系数图 (coefplot) | 跨模型比较系数 | 标注 0 参考线、区分模型颜色 |
| 边际效应图 | 交互项或非线性模型 | 加 95% CI 带 |
| 事件研究图 | DID 平行趋势 + 动态效应 | 标注处理时点竖线、0 参考线 |
| RDD 图 | 断点处的跳跃 | 分桶均值 + 局部多项式拟合 |
| 莫兰散点图 | 空间自相关 | 四象限 + 拟合线 |
| IRF 脉冲响应图 | VAR/DSGE 冲击传导 | 标注 0 参考线、加 CI 带 |
| 主题频次图 | 质性编码分布 | 条形图从高到低排序 |

### 6.3 图注模板

```
图 1：[图的核心信息]。注：横轴为 [X]，纵轴为 [Y]。
阴影区域表示 95% 置信区间。数据来源：[来源]。
```

## 7. 论文段落模板

### 7.1 基准回归

> "表 2 报告了 [Y] 与 [X] 的基础回归结果。列(1)仅加入核心解释变量，[X] 的系数为 正/负，在 [1%/5%/10%] 水平上与 [Y] 显著 正/负 相关。列(2)加入个体层面控制变量后，系数方向 不变/反转。列(3)进一步加入年份和地区固定效应，[X] 的系数为 正/负 且在 [1%/5%/10%] 水平上统计显著。这一效应不仅统计显著，而且具有实质经济含义：[X] 如何怎么变化的 [Y]。"

### 7.2 非线性模型

> "由于因变量为二元变量，表 3 报告了 Logit 模型的平均边际效应 (AME)。在完整模型中，[X] 的 AME 为 正/负 且在 [1%/5%/10%] 水平上统计显著。该估计意味着，在其他条件不变时，[X] 从 0 变化到 1 时，[Y=1] 的预测概率平均 提高/降低 若干个百分点。该效应与列(1)和列(2)的估计方向一致，表明核心结论对模型设定不敏感。"

### 7.3 面板固定效应

> "表 4 报告了面板数据回归结果。Hausman 检验在 [1%/5%/10%] 水平上拒绝随机效应一致有效的原假设，以下分析以个体固定效应模型为准。列(2)的组内估计显示，在控制了个体不可观测的时不变异质性后，[X] 的系数为 正/负 且在 [1%/5%/10%] 水平上统计显著。这一系数反映的是同一个体内 [X] 随时间的变化如何影响 [Y] 的变化——而非不同个体之间的截面差异。与混合 OLS 估计相比，固定效应估计的系数绝对值 有所减小/增大，提示个体层面的时不变异质性 夸大/掩盖 了部分关联。"

### 7.4 工具变量

> "表 5 报告了工具变量估计结果。第一阶段中，工具变量 [IV] 与内生变量 [X] 在 [1%/5%/10%] 水平上显著 正/负 相关，第一阶段 F 统计量超过弱工具变量检验的经验阈值（F > 10），排除弱工具变量的顾虑。第二阶段估计显示，在纠正内生性偏误后，[X] 对 [Y] 的因果效应仍为 正/负 且在 [1%/5%/10%] 水平上统计显著，但系数绝对值 低于/高于 OLS 估计，与遗漏 正/负 相关变量的预期一致。Hansen J 检验不能拒绝工具变量外生性的原假设——但排除限制的可信度最终依赖理论论证而非统计检验。"

### 7.5 DID

> "表 6 报告了双重差分估计结果。处理组与对照组在处理前趋势平行（事件研究图中处理前的交互项均未达显著，联合 F 检验不拒绝平行趋势原假设），支持平行趋势假设。核心交互项 Treat×Post 的系数为 正/负 且在 [1%/5%/10%] 水平上统计显著，表示政策实施后，处理组的 [Y] 相对于对照组平均 提高/降低 了一定幅度。安慰剂检验——将处理时间提前两期——未检测到显著的处理效应，进一步增强了因果解释的可信度。"

### 7.6 多期 DID (交错处理)

> "由于处理时点存在交错——个体在不同年份进入处理——传统 TWFE 估计量可能因负权重和异质性处理效应而产生偏误。表 6 额外报告了三种异质性稳健估计量：Callaway-Sant'Anna（csdid）、Borusyak-Jaravel-Spiess 插补法，以及 Sun-Abraham 交互加权事件研究法。三者的估计结果与 TWFE 方向一致，但量级略有差异。动态处理效应图显示，处理效应在政策实施后逐步显现，约在第 2 期达到峰值。"

### 7.7 RDD

## 12. 避免事项总表

| 禁止 | 原因 | 替代做法 |
|------|------|---------|
| "边缘显著" / "marginally significant" | p 值二分逻辑下不存在"边缘" | 报告 exact p 并如实讨论 |
| 只报 Logit/Probit 系数 | 非线性模型系数无直观含义 | 报告 AME 或预测概率变化 |
| 系数下方放标准误 | SE 不可用于显著性判断 | 放 t/z 统计值 |
| "X 影响 Y"（无因果设计证据） | 相关 ≠ 因果 | "相关""预测""伴随变化" |
| 选择性报告显著模型 | 发表偏倚 | 报告全部模型，无论显著性 |
| 不报告第一阶段 | IV 可信度的核心证据 | 第一阶段 F + 系数 + 方向 |
| 把控制变量的显著性当理论贡献 | 控制变量的系数无因果解释 | 只讨论核心解释变量 |
| 固定效应模型说"控制所有遗漏变量" | FE 只控制时不变遗漏变量 | 说明"控制了不随时间变化的个体异质性" |
| 表格中混用 SE 和 t 统计值 | 读者困惑 | 统一一种格式 |
| 无图注或含义不明的图注 | 图形不可独立理解 | 图注说明核心发现 |
