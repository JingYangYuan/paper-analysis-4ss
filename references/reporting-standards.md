# 结果报告规范

## 1. 定量结果

正文报告：

- 方向：正向、负向、不显著。
- 量级：系数、AME、预测概率变化、IRR。
- 不确定性：正文可报告 CI 或 exact p；回归表默认在系数下方放 t/z 统计值。
- 样本和模型：N、控制变量、固定效应、标准误。

数值格式：

- 描述统计、系数、边际效应、t/z 统计值、比例：保留 3 位小数。
- 样本量、频数、自由度：保留整数。

避免：

- 将 p 值当作效果大小。
- 把不显著写成“边缘显著趋势”。
- 只解释 logit 系数。
- 选择性报告显著模型。

## 2. 中文表注模板

“注：括号内为 t/z 统计值。所有模型均控制 [控制变量]；列 [x] 加入 [固定效应]。标准误按 [层级] 聚类。* p < 0.05，** p < 0.01，*** p < 0.001。”

## 3. 回归结果段模板

“表 X 报告了 [Y] 对 [X] 的基础回归结果。列 1 仅控制 [基础变量]，列 2 加入 [控制变量]，列 3 进一步加入 [固定效应]。在完整模型中，[X] 的系数为 [b]，括号内 t/z 统计值为 [stat]，说明在其他变量保持不变时，[解释量级]。该结果在 [稳健性] 后保持一致/发生变化。”

## 4. 质性结果段模板

“质性材料显示，[主题] 是解释 [现象] 的关键机制。该主题并非单一态度陈述，而是由 [编码 A]、[编码 B] 和 [编码 C] 共同构成。受访者 [匿名编号] 的表述体现了这一机制：[短引文]。这一发现说明 [解释]，并与定量结果中的 [发现] 形成 [收敛/互补/分歧]。”

## 5. 混合方法段模板

“综合表 X 将回归结果与质性主题并置。定量分析发现 [结果]；质性编码进一步显示 [主题]，说明 [机制]。两类证据在 [方面] 上收敛，但在 [方面] 上存在分歧，提示 [边界条件或测量问题]。”
